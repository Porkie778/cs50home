HTML

HTML Structure and Tags
HTML, or Hypertext Markup Language, structures web content with tags. Essential tags include <html>, <head>, <title>, and <body>. Tags like <h1> to <h6> represent headers, while <p> denotes paragraphs.

Attributes
Tags may have attributes to provide additional information, such as lang="en" in <html>. Attributes are used to specify details about elements.

Hierarchy
HTML tags create a nested structure that helps browsers interpret content organization. Correct hierarchy and nesting of tags are essential for proper display and SEO.

CSS

Selectors and Properties
CSS, or Cascading Style Sheets, controls styling and layout. Basic selectors like id and class allow applying styles to elements, while properties like color, font-size, and margin define visual presentation.

Box Model
CSS uses the box model, where each element is considered a box comprising margins, borders, padding, and the content area. Adjusting these properties is essential for layout control.

Frameworks
Frameworks like Bootstrap simplify design by offering pre-styled components and grids, reducing the need to write extensive CSS.

JavaScript

Variables, Conditionals, and Loops
JavaScript adds interactivity. Variables store data, conditionals (if-else) control flow, and loops (for, while) execute repetitive tasks.

DOM Manipulation
JavaScript accesses and modifies the Document Object Model (DOM) to dynamically update page content. Functions like document.querySelector() target elements, while events such as click trigger responses to user actions.

Events
Event listeners (addEventListener) detect user actions, enabling interactive experiences, like responding to button clicks or form submissions.
